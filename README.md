# 블록체인 기반 의료 장비 관리 시스템 (Capstone)

졸업작품으로 개발한 **블록체인 기반 의료 장비 관리 시스템**입니다.  
의료기관 내 장비의 등록, 이력 추적, 점검/수리 기록, 사용자 권한 관리를 투명하고 신뢰성 있게 운영하는 것을 목표로 합니다.

---

## 1. 프로젝트 개요

의료 장비는 환자 안전과 직결되기 때문에,
- 장비의 상태 정보가 정확해야 하고,
- 점검/수리 이력이 위변조 없이 관리되어야 하며,
- 누가 언제 어떤 조치를 했는지 추적 가능해야 합니다.

이 프로젝트는 위 요구사항을 만족하기 위해 **블록체인(분산원장)**을 도입하여 장비 관리 이력을 신뢰성 있게 저장하고, 웹 애플리케이션을 통해 현장에서 쉽게 관리할 수 있도록 설계했습니다.

---

## 2. 핵심 목표

- 의료 장비 데이터의 **무결성(Integrity)** 확보
- 점검·수리·교체 기록의 **투명한 추적성(Traceability)** 제공
- 병원 내 역할별 접근 제어를 통한 **운영 효율성** 향상
- 기존 수기/분절 시스템 대비 **관리 자동화** 수준 개선

---

## 3. 주요 기능

- **장비 등록 및 조회**
  - 장비 ID, 모델명, 제조사, 도입일, 위치 등 기본 정보 관리
- **점검/수리 이력 관리**
  - 정기 점검 결과, 고장 접수, 수리 내역, 부품 교체 이력 저장
- **상태 모니터링**
  - 사용 가능 / 점검 필요 / 수리 중 / 폐기 예정 상태 구분
- **블록체인 기록**
  - 중요한 이벤트(등록, 점검 완료, 수리 완료 등)를 트랜잭션으로 저장
- **권한 기반 접근 제어**
  - 관리자, 엔지니어, 의료진 등 역할별 기능 제한
- **감사 로그(Audit Log)**
  - 누가 어떤 데이터를 수정·승인했는지 이력 확인

---

## 4. 기대 효과

- 기록 위변조 위험 감소
- 병원 내 장비 관리 책임 추적 용이
- 인증/감사 대응 시 신뢰성 높은 데이터 제공
- 장비 유지보수 주기 최적화를 통한 운영 비용 절감
